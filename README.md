### INSTALLATION

* apt-get update

* apt-get upgrade -y

* pkg install git -y

* git clone https://github.com/Tech-with-loser/install-keys

* cd install-keys

* chmod +x *

* bash install.sh

### UNINSTALL KEYS

* cd install-keys

* bash remove.sh
